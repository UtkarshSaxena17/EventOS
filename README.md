# 🚀 EventOS

**EventOS** is a fast, lightweight, and unified Hackathon & Event Management Platform designed to streamline the entire event experience. 

It brings organizers, participants, and judges together into a single cohesive ecosystem. Say goodbye to messy spreadsheets and disjointed tools—EventOS offers real-time syncing and role-specific portals to handle everything effortlessly.

## ✨ Key Features

* **👩‍💻 Participant Hub:** A dedicated portal for hackers. Join teams, submit project links (GitHub/Demo), and track live global leaderboards.
* **⚖️ Judge Sandbox:** A blind-grading interface offering objective scoring rubrics, dynamically linked to the main leaderboard.
* **👑 Admin Dashboard:** Ultimate event control. Add judges, manage project rooms, monitor live statistics, ban users, and broadcast global announcements instantly.

## 🛠️ Tech Stack

* **Frontend:** Vanilla HTML/CSS with modular Vanilla JavaScript (SPA structure).
* **Backend:** Node.js paired with an Express REST API.
* **Database:** MySQL for robust and structured data management.
